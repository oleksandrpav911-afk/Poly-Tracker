# הוספת מספר ארנקים למעקב

המערכת תומכת כעת במעקב אחר מספר ארנקים במקביל, עם התראות לאותה קבוצה בטלגרם.

## איך להוסיף ארנק חדש

1. פתח את הקובץ `telegram_notifier.py`
2. מצא את הרשימה `WALLETS` (בשורה ~18)
3. הוסף ארנק חדש כך:

```python
WALLETS = [
    {
        'address': '0x16b29c50f2439faf627209b2ac0c7bbddaa8a881',
        'name': 'SeriouslySirius'
    },
    {
        'address': '0x...',  # כתובת הארנק החדש
        'name': 'שם הארנק'    # שם שתרצה לראות בהתראות
    },
]
```

4. שמור את הקובץ

## מה קורה אחרי הוספת ארנק?

- **התראות טלגרם**: כל הארנקים ישלחו התראות לאותה קבוצה בטלגרם, עם שם הארנק בכל הודעה
- **מעקב פוזיציות**: כל הארנקים ייבדקו אוטומטית לשינויים בפוזיציות
- **דשבורד**: בדשבורד תוכל לבחור איזה ארנק לראות, או לראות את כל הארנקים יחד

## הערות חשובות

- כל הארנקים משתמשים באותה קבוצה בטלגרם (TELEGRAM_CHAT_ID)
- כל הארנקים משתמשים באותם כללי סינון (MIN_TRADE_SIZE_USDC, ALLOWED_SPORTS)
- כל הארנקים נשמרים באותו קובץ `processed_trades.json` (עם מפתח ייחודי לכל ארנק)
- פוזיציות נשמרות ב-`positions_snapshot.json` עם מפתח ייחודי לכל ארנק

## קבצים שעודכנו

- `telegram_notifier.py` - הגדרת רשימת הארנקים
- `telegram_monitor.py` - מעקב אחר כל הארנקים
- `position_tracker.py` - מעקב פוזיציות לכל הארנקים
- `dashboard.py` - תצוגה של ארנק ספציפי או כל הארנקים

## דוגמה לשימוש

אחרי הוספת ארנק חדש, פשוט הפעל מחדש את:
- `telegram_monitor.py` - למעקב עסקאות
- `position_tracker.py` - למעקב פוזיציות
- `dashboard.py` - לדשבורד

כל הארנקים יעבדו אוטומטית!
